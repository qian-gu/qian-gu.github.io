Background

工作太忙博客荒废了 4 年多时间，现在重新拾起，利用周末时间充电学习希望能坚持下去，立个 flag：

争取以后每个周末能学习一点新内容，总结记录下来。

写这个学习笔记的原因是我对很多 Python 概念和用法一知半解，所以想重新学习 Python 的知识，顺便记录下来方便以后温故而知新。
